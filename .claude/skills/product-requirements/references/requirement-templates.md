# Product Requirements Templates

Ready-to-use templates for different product types to help clients articulate their needs.

## Template 1: Business Productivity Tool

**Product Name:** [Name]

**Core Purpose:**
- Helps [user type] accomplish [main task]
- Solves the problem of [pain point]
- Replaces/improves [current solution]

**Target Users:**
- Primary: [job role, team size, industry]
- Secondary: [additional user types]
- Technical level: [beginner/intermediate/advanced]

**Must-Have Features:**
1. [Feature] - allows users to [action]
2. [Feature] - enables [capability]
3. [Feature] - provides [benefit]

**Key User Flow:**
1. User logs in and sees [dashboard/home screen]
2. User navigates to [section] to [action]
3. User creates/edits [object]
4. System processes and displays [result]
5. User can [next action]

**Success Metrics:**
- [Number] active users within [timeframe]
- [Percentage] user retention rate
- [Metric] indicating engagement
